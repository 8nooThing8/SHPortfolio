TGA projects
No Broom For Error - Project 1
No Brom For Error was made in a XX week period on half-time and was the first project at The Game Assembly, the reference game was "Race The Sun" by Flippfly LLC.

The main thing i created for this game was a tool to create a sequence of animated obstacles like trees that fall down to try to hit you,
I also created the moon functionality to sync up the time left. I also created a tool to move clothes on a clothes line using a bezier curve.
Hole-É-Mole - Project 2
Hole-É-Mole was made during a XX week period on half-time and was a mobile game made for Android, Reference game was "Lara Croft GO" by Square Enix Montréal.

The main function i made was the moving platforms that worked using the tiling system we created, something that was important was that adding a moving platform and creating the path would be easy and intuitive.
I created most of the Menus in the game, Main menu, level select and settings.
TRAUM - Project 3
TRAUM was made during a 12 week period on half-time and the reference game was "Limbo" by Playdead.

I was the creator of the systems for most moving things. Things like sprite sheet animator, Video player and spritesheet exporter from Unity meta files.
TRAUM became the winner of 3 of 4 total nominated awards at Swedish game awards.
Winner: Game Of The Year, Best Narrative, Best Diversity.
Nominee: Best Audio
Golem's Quest: Crystal Keep - Project 4
Golem's Quest was made duing a XX Week period on half time and the reference game was "Death's Door" by Acid Nerve.

The main thing i made in this project and had the most impact was the shadows, this was very fun and i learned alot from this.
I created the specilised shaders like the enemy damage shader to make more cracks depending on the health the enemy has left.
