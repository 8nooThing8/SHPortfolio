Intro/Idéa
While deciding on an idea we were thinking the theme built to scale as in make things smaller or bigger to make them fit like everyone else. An idéa came to interperate built to scale as something that someone made to climb also called scale in some instances and that is how a grapple climbing game was decided. I was the only one in the group who had done some graphics so i became the graphics team for the project.



Rope Rendering
We need a rope to use as the grapple string and we started by using line renderer in unity which worked well until the line curved around objects and the connection points was very few which made the line phase through these objects. To fix this i created a 3D version of the line renderer that takes in a mesh and still uses line renderer as a base to know where the line will start and end. I also made the line be a bit more strict on where it would connect.



Camera Controller
The camera controlls and effects was made by me. Making the camera zoom out as you go faster made the most effect. Yolu could then feel that you are going fast and felt very good in general.



Closing Thoughts
Grapple Ascent was a very fun game to work on and we made a very cool product in the end its also very fun and satisfying
