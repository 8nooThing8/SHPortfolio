Tree Fall Sequence Editor
This was the first project at The Game Assembly and i was assigned the task to make a hopefully easy tool for making falling trees in this endless runner game. The tool was made with unity's editor tools which made it possible to do things like only displaying information when a flag was set which is not possible in the default properties editor. In the end i made a tool that could do this, the only problem was that it was not very user friendly since i had not made many tools that others had to use which made it hard to use and thus not very many of these instances can be seen in the game. It was a learningful experince and learnetmore about the unity tool.

Moon Timer
We wanted a fun way to show how much time is left so we decided to do it the same way race the sun does it, our reference game. In race the sun, like the name sugests you have to keep racing the sun while it keeps going down and when you get more time it rises again. Since we had a haloween theme and a witch as the main character we made the sun be a moon instead. My solution to this was pretty simple, have a start position and have an end position and lerp smoothly between these with (Time passed รท max time). This made a pretty nice and smooth move from highest point to lowest point. One problem this caused was the moon in the final product was too close and can sometimes be seen clipping into the ground, the solution move it further back.

Bezier Curving
In our project we wanted to make a clothes line that moves forwards and back. The easiest way to do this like i did was to just make a simple bezier curve between two vector3's. A bezier curve could be made very complicated with a bunch of math functions but that would be way too indepth for this project so i just decided to use multiple Vector3 lerps to smoothly go between these points.
